# Homepage (tbasystent.pl) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a single-page SSR marketing homepage at `/` that converts Ania (cleaning business owner) to beta signup within 5 seconds on mobile.

**Architecture:** Laravel Blade + Tailwind CSS + Alpine.js (installed separately for the public layout — Filament owns its own Alpine instance under `/admin`). The page is static SSR content with one small Alpine widget (branża switcher). No Livewire. No Vue.

**Tech Stack:** PHP/Laravel 11, Blade, Tailwind CSS (existing), Alpine.js (to be installed), Vite, Pest (tests)

**Spec:** `docs/superpowers/specs/2026-05-16-homepage-design.md`
